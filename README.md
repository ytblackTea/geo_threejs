# geo_threejs
threejs绘图
需求分析：
期望效果：

线条颜色可变（支持光线跑动）
球体可以自动旋转（支持鼠标拖拽）
输入经纬度可自定从起点连接到结束点。（自动连接线）
鼠标经过可以显示不同国家的地理位置信息（把国家的领土变色）
可以根据数据状态实时变更线条颜色以及领土颜色。
